Token information

  • Token Name: Camino (CAM)
  • Token Supply: 1,000,000,000

Token distribution

Camino (CAM) is the native token of the Camino network. It is used to secure the network through staking or pay for fees.

Camino (CAM)

The Camino native utility token is used for gas price and transactions fees. In a second phase Camino will also be used by enduser for payment.

Stable Coin

We are supporting USDollar stable coins. In a second phase we will also support stable EURO coins.
